# PortSwigger Web Security Academy Labs

This repository contains my learning journey through PortSwigger Web Security Academy.

## Topics Covered

* SQL Injection
* Cross-Site Scripting (XSS)
* Cross-Site Request Forgery (CSRF)
* Access Control
* Authentication
* Server-Side Request Forgery (SSRF)
* XML External Entity (XXE)
* JWT
* File Upload Vulnerabilities
* API Testing
* GraphQL
* Web Cache Poisoning
* HTTP Request Smuggling

## Objectives

* Understand web application vulnerabilities.
* Practice exploitation techniques in a legal lab environment.
* Document methodologies and payloads.
* Build a knowledge base for VAPT and Bug Bounty hunting.

## Disclaimer

All testing was performed in authorized lab environments provided by PortSwigger Web Security Academy.
